## 🚀 About the Project
Ever tried guessing a number your friend had in mind? This is exactly that — but with a computer! The Number Guessing Game is a fun console-based Java game where you guess a randomly picked number before your attempts run out. The faster you guess, the higher your score!

---

## ✨ Key Features
- 🎯 3 Difficulty levels – Easy, Medium, and Hard
- 🔢 Random number generation within difficulty range
- 🔁 Play Again option for multiple rounds
- 🏆 Scoring system — fewer attempts means more points
- 📊 Session summary with best score and average score
- 📋 Round-wise score table
- ✅ Input validation – handles wrong type and out of range input
- 👤 Player name personalization

---

## 🛠️ Tech Stack
- Java
- OOP (Classes and Methods)
- Exception Handling (try-catch)
- Collections (ArrayList)

---

## ▶️ How to Run

**Step 1 – Compile**
javac NumberGuessingGame.java

**Step 2 – Run**
java NumberGuessingGame
